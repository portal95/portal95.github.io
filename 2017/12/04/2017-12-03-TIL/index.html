자바스크립트의 4가지 데이터타입

/숫자 - 1,4.5
문자 - “HI”,’Samething’
불리언 - True, False
null/

-각 타입은 타입마다의 연산을 가지고있음.
-모든연산은 같은 타입끼리만 할수있다.

*자바스크립트는 자동으로 타입을 바꿔서 연산해주기도 한다.
(“안녕”+20–>”안녕20” // 153 + “” –> “153” // 153 - “” –> 153)

//typeof() 타입 알려주는 함수

주석
“//“

/ 와 / 사이에 여러줄의 주석 입력가능.

변수

var a; 로 정의

“=” 은 수학의 같다의 의미가 아니고 오른쪽의 값을 왼쪽 변수에 넣어준단 뜻

Class, Object와 Instance
값을 저장하면 특정 객체(Object) 또는 데이터 타입의 인스턴스가 생성됨
비슷한 객체의 묶음을 클래스라고 부른다
자동차가 클래스라면 도로의 차들 각각이 자동차의 인스터스라고 생각할수 있다.
각각의 숫자와 문자열도 정확하게는 숫자형, 문자형 인스턴스

객체의 속성
properties
객체들은 고유한 속성을 가지며 이를 속성(property)이라고 부릅니다.
속성값은.(dot operator)를 통해 확인할 수 있습니다.
//철수의 키 –> 철수.키
‘hello’.length;

methods
객체들은 지정된 작업을 수행할수 있는데, 이를 메소드라고 합니다.
유사한 개념으로 함수가 있습니다.
객체의 함수를 메소드라고 볼 수도 있습니다.
메소드를 실행하면 대부분의 경우 값이 바뀜.(항상은 아님)

//철수야 먹어 –> 철수.먹어()
//철수야 밥 먹어 –> 철수.먹어(‘밥’);
//철수야 지금 밥 먹어 –> 철수.먹어(‘밥’,now);
‘hello’.toUpperString();

내장 객체 사용하기
편리하게 사용하는 내장(built-in)객체들이 존재함.
console.log(“hello”); 콘솔 객체의.로그 메소드

많이 사용하는 객체
math. 수학용 객체.
math.random() 0에서 1사이의 난수
math.floor() 내림
math.ceil() 올림
math.round() 반올림
